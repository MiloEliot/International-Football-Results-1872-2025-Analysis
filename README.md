# International Football Results (1872-2025) PostgreSQL Analysis and Machine Learning

This project explores the International Football Results (1872-2025) from Kaggle using Postgresql. 

It demonstrates: 
- Basic SQL commands 
- Common table expression
- Advanced SQL features (function, anonymous do block, window function)
- PostgreSQL VIEW
- Exploratory data analysis and visualization 
- Data processing for machine learning (standarization of the numerical data and encoding the categorical data)
- Prediction of the match outcome with RandomizedForestClassifier, XGBoostClassifier, and CatBoostClassifier
- Hyperparameter tuning using RandomizedSearchCV, HalvingGridSearchCV, and BayesSearchCV

## Dataset
The original dataset can be found at: [Kaggle](https://www.kaggle.com/datasets/martj42/international-football-results-from-1872-to-2017/data). 


## Repository Structure
- `Data` - The downloaded CSV files
- `SQL` - SQL script used to build the database and the schema
- `International_Football_Results_Analysis` - SQL queries, exploratory data analysis, and machine learning
